# 剪羊毛

> 原文：<https://www.javatpoint.com/computer-graphics-3d-shearing>

它是物体形状的变化。它也被称为变形。在 2D 的情况下，变化可以是 x 方向或 y 方向，或者两个方向都有。如果剪切发生在两个方向，物体将会变形。但是在 3D 中，剪切可以发生在三个方向。
