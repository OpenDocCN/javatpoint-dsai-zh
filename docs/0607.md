## 算法:

**第一步:**把 x =0，y =r 放在等式 2 中
我们有 p=1-r

**步骤 2:** 重复步骤，同时 x ≤ y
绘制(x，y)
If (p < 0)
然后设置 p = p + 2x + 3
否则
p = p + 2(x-y)+5
y =y - 1(结束 if)
x =x+1(结束循环)

**步骤 3:** 结束
