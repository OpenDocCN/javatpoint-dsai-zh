# MATLAB 正弦波图

> 原文:[https://www.javatpoint.com/matlab-sine-wave-plot](https://www.javatpoint.com/matlab-sine-wave-plot)

**目的:**绘制频率为 1KHz 的正弦波。

**例:**我们来生成一个简单连续的类正弦信号，频率 FM=1KHz。为了使其在绘图时作为重复信号出现，使用 fs=500KHz 的采样率。

```
fs= 500e3;
f= 1000;
nCyl=5;
t=0:1/fs:nCyl*1/f;
x=sin(2*pi*f*t);
plot(t,x)
title ('Continuous sinusoidal signal')
xlabel('Time(s)');
ylabel('Amplitude');

```
