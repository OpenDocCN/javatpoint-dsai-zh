# MATLAB 网格()

> 原文:[https://www.javatpoint.com/matlab-3d-mesh](https://www.javatpoint.com/matlab-3d-mesh)

网格创建由 X、Y 和 Z 指定的线框参数化曲面，颜色由 c 指定。

### 句法

```

mesh(X,Y,Z)
mesh(Z)
mesh(...,C)
mesh(...,'PropertyName',PropertyValue,...)
meshc(...)
meshz(...)
h = mesh(...)
h = meshc(...)
h = meshz(...)

```

### 例子

```

z=-5/(1+x^2+y^2 )
|x|≤3,|y|≤3
x=linspace (-3, 3, 50);
y=x;
[x, y]=meshgrid(x, y);
z=-5./(1+x.^2+y.^2);
mesh(z)

```
