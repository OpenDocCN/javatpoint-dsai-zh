# MATLAB 轮廓()

> 原文:[https://www.javatpoint.com/matlab-2d-contour](https://www.javatpoint.com/matlab-2d-contour)

等高线图显示了矩阵 z 的等值线。

### 句法

```

contour(Z)
contour(Z,n)
contour(Z,v)
contour(X,Y,Z)
contour(X,Y,Z,n)
contour(X,Y,Z,v)
contour(...,LineSpec)
[C,h] = contour(...)

```

### 例子

```

z=-1/2 x^2+xy+y^2
|x|≤5,|y|≤5.
r= -5: .2:5;
[X, Y]=meshgrid (r, r);
Z=-.5*X.^2+X.*Y+Y.^2;
cs= contour (X, Y, Z);
clabel (cs)

```
